# Azucat

## About
標準入力をブラウザに出力します。
WebSocketが動作するブラウザで動作します。
Ruby 1.8.7以上で動作確認済みです。

## Install
    $ gem install bundler
    $ bundler

## Example
    $ tail -f foo.log | ./bin/azucat
